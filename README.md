# Breast Cancer Detection using ANN

This project predicts whether a breast tumor is Benign or Malignant using an Artificial Neural Network (ANN).

## Dataset
Breast Cancer Wisconsin (Diagnostic) Dataset

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow / Keras

## Workflow
1. Load and clean dataset
2. Perform Exploratory Data Analysis
3. Feature scaling using StandardScaler
4. Build ANN model
5. Train model
6. Evaluate accuracy and confusion matrix

## Model Architecture
- Dense (64, ReLU)
- Dropout (0.2)
- Dense (32, ReLU)
- Dropout (0.2)
- Dense (16, ReLU)
- Output Layer (Sigmoid)

## Results
- High classification accuracy on test data
- Successfully predicts Benign / Malignant tumors
